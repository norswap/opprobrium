# Opprobrium

## Installation

Tooling required:

- [Foundry](https://github.com/gakonst/foundry)
- Make
- Node.js & [PNPM](https://pnpm.io/) (`npm install -g pnpm`)
  - Tested with Node v16.16.0

## Commands

- `make setup` - initialize libraries and npm packages
- `make update-deps` - update node dependencies
